# 🥖 PadariaSystem

> Sistema de gerenciamento de padaria desenvolvido em Java para a disciplina de **Orientação a Objetos**, com controle de usuários, produtos, clientes e vendas via terminal.

## Funcionalidades

- Autenticação de usuários com troca de senha no primeiro login
- Controle de acesso por perfil (Administrador, Atendente e Padeiro)
- Cadastro e gerenciamento de produtos (Pão, Bebida, Doce e Salgado)
- Cadastro de clientes
- Registro de vendas com cálculo automático do valor total

## Perfis de Acesso

| Perfil | Permissões |
|---|---|
| Administrador | Clientes, Produtos, Vendas e Usuários |
| Atendente | Clientes, Produtos e Vendas |
| Padeiro | Produtos |

## Tecnologias

- Java
- Paradigma orientado a objetos (herança, classes abstratas, encapsulamento)
